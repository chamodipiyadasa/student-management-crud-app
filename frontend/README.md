# Frontend (React + Vite)

This is a minimal React + Vite frontend for the Student Management demo.

Quick start (from repo root):

```bash
cd "./frontend"
npm install
npm run dev
```

The app expects the backend API at `VITE_API_BASE_URL` (default `http://localhost:5000`). Example `.env`:

```
VITE_API_BASE_URL=http://localhost:5000
```
